## 🎨 Customization Guide

### Edit Your Information

1. **Update Hero Section** (in `index.html`):
   - Change the title and subtitle
   - Update button text if needed

2. **Update About Section**:
   - Replace the placeholder text with your bio
   - Update statistics numbers

3. **Customize Skills**:
   - Modify skill categories and tags
   - Add or remove skills as needed

4. **Update Projects**:
   - Replace project titles and descriptions
   - Update technologies and links
   - Add your own project details

5. **Update Education**:
   - Keep NJIT information (as required)
   - Add/modify your specific details
   - Update graduation dates and GPA

6. **Update Awards**:
   - Replace with your actual accolades
   - Modify years and descriptions

7. **Update Contact**:
   - Replace email address
   - Update phone number
   - Change location
   - Add your social media links

### Color Customization

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #007bff;      /* Main brand color */
    --secondary-color: #0056b3;    /* Secondary color */
    --accent-color: #ff6b6b;       /* Accent color */
    --dark-bg: #0f1419;            /* Dark background */
    --light-bg: #ffffff;           /* Light background */
    /* ... more variables ... */
}
```

## 🚀 How to Use

### Option 1: Open Locally
1. Extract all files to a folder
2. Open `index.html` in a web browser
3. Navigate through sections using the menu

### Option 2: Deploy Online
The portfolio can be deployed to:
- **GitHub Pages** - Free hosting for static sites
- **Netlify** - Drag and drop deployment
- **Vercel** - Optimized for static sites
- **Any web hosting service** - FTP/upload the files

### Steps for GitHub Pages:
1. Create a GitHub repository
2. Upload the three files (index.html, styles.css, script.js)
3. Go to repository Settings → Pages
4. Select main branch as source
5. Your site will be live at `https://yourusername.github.io/portfolio`
